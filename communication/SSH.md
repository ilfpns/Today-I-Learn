> **Secure Shell Protocol**
> 

---

⇒ 네트워크 프로토콜 중 하나로 컴퓨터와 컴퓨터가 인터넷과 같은 Public Network를 통해 서로 안정적인 통신을 할 때 사용됨

 **ㅤ**

**SSH의 기능**

- 네트워크 상의 다른 컴퓨터로 로그인
- 원격 시스템에서 명령을 실행
- 다른 시스템으로 파일을 복사

**ㅤ**

**특징**

- SSH는 데이터를 암호화하여 전송한다
- 네트워크 상에서 도청이나, 데이터 조작을 방지한다
- 22번 포트를 사용한다

**ㅤ**

⇒ telnet은 SSH와 같은 역할을 하지만 보안이 부족하여 유출 위험이 있음

**ㅤ**

> **통신 과정**
> 

---

⇒ SSH는 client와 server로 나뉘어지는데, 서로 올바른 노드인지 인증하기 위해 Public Key, Private Key를 가지고 있다.

⇒ 이를 비대칭키 암호화 방식이라 한다
