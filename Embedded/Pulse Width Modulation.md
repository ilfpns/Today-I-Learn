# Pulse Width Modulation이란?

→ 디지털 신호 중 특정한 형태를 띈 신호를 일컷음

> 디지털 신호를 마치 아날로그 신호처럼 사용할 수 있음
> 

# Duty cycle (듀티비)

→ HIGH 신호와 LOW신호의 비율을 표시함
